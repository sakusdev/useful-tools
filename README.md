# Useful Tools

Modern modular utility toolbox.

## Stack

- Vite
- React
- TypeScript
- TailwindCSS
- GitHub Pages

## Goals

- Modular architecture
- Frontend only
- PWA ready
- Lazy loading
- 50+ utility tools
